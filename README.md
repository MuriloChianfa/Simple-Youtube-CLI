Installing the Entity Framework
```bash
$ Install-Package Microsoft.EntityFrameworkCore.Sqlite
```

To create database: 
```bash
$ Install-Package Microsoft.EntityFrameworkCore.Tools
$ Add-Migration InitialCreate
$ Update-Database
```
